Consulta de Calificaciones

Escribe tu número de matrícula tal como aparece en la lista del plantel.

Consejo: también puedes presionar Enter para buscar.

¿Cómo cargo/edito la lista de alumnos?

Abre este archivo y edita el bloque DATA de abajo. Cada entrada usa la matrícula como clave y contiene nombre y calificacion. Ejemplo:

{
  "202500101": { "nombre": "Hortilla-Méndez, Tania Arleth", "calificacion": 85 },
  "202500102": { "nombre": "Cruz Contreras, Ángel Gabriel", "calificacion": 74 }
}

Guarda los cambios y vuelve a subir/publicar el sitio (por ejemplo, en GitHub Pages o Netlify). Si prefieres, puedes usar el importador CSV rápido aquí debajo.

Pega un CSV con columnas: matricula,nombre,calificacion. Esto no guarda nada en un servidor: solo reemplaza DATA temporalmente en tu navegador para probar.
